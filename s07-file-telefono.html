File

File anche sul telefono

Lo smartphone gestisce file come un computer: foto, video, documenti e app convivono in un file system strutturato.

Dove si trovano

Android: cartelle Download, DCIM (foto), Documenti. Ogni app ha la propria cartella in /Android/data/. iOS: app File → Nel mio iPhone. I file dentro le singole app non sono sempre visibili da File.

📁

File anche sul telefono

  • I file sono organizzati in cartelle come su un PC, anche se l'interfaccia lo nasconde.
  • Foto in DCIM, download nella cartella Download, documenti in Documenti o nell'app che li ha aperti.
  • iOS limita l'accesso diretto: ogni app ha il proprio contenitore e i file non sono condivisi liberamente.

File

App File

Il file manager permette di navigare la memoria, spostare, copiare e cancellare file senza passare dalle singole app.

Come usarla

Android: "File" (Google) o "Gestore file" del produttore. iOS: app "File". Entrambi mostrano cloud integrato (Drive, iCloud) e memoria locale. I file recenti appaiono in cima alla lista.

📂

App File

  • Permette di copiare, spostare, rinominare e cancellare file senza entrare nelle singole app.
  • Mostra sia la memoria locale sia i cloud collegati (Drive, Dropbox, iCloud) in un'unica interfaccia.
  • Utile per trovare file scaricati o inviati via WhatsApp che non sai dove sono finiti.

Download e documenti

I file ricevuti via web o email si accumulano nelle cartelle Download e Documenti: vanno revisionati periodicamente.

📥

Download e documenti

  • I download dal browser finiscono in Download; quelli dalle email nella cartella mail o in Download.
  • I documenti PDF e Office salvati si trovano in Documenti o nella cartella dell'app usata per aprirli.
  • La cartella Download si riempie di file dimenticati: revisionala ogni 2–4 settimane.
  • Sposta i documenti importanti in una cartella nominata e nel cloud per averli anche da PC.

Foto, video e allegati

Foto dalla galleria, video registrati e allegati da chat occupano la maggior parte dello spazio: gestirli riduce i consumi.

🖼️

Foto, video e allegati

  • Foto e video in DCIM sono i maggiori consumatori di spazio: video 4K occupano 300–500 MB al minuto.
  • Gli allegati ricevuti via WhatsApp e Telegram si duplicano in download locali anche se già nel cloud.
  • Usa Google Foto o iCloud per sincronizzazione automatica, poi cancella le copie locali per liberare spazio.
  • Scarica su PC i contenuti a cui tieni prima di cancellare dal telefono.

Cloud e backup

Il cloud archivia file accessibili da qualsiasi dispositivo; il backup automatico protegge i dati in caso di perdita o furto.

☁️

Cloud e backup

  • Cloud storage (Drive, iCloud, OneDrive, Dropbox) sincronizza i file in tempo reale su tutti i dispositivi.
  • Il backup automatico del telefono include app, impostazioni, messaggi e foto.
  • Controlla che il backup sia attivo e aggiornato: Impostazioni → account cloud → Backup.
  • Lo spazio gratuito (15 GB Google, 5 GB iCloud) si esaurisce: valuta un abbonamento se il telefono è il tuo strumento principale.

Spazio e cache

La cache velocizza le app ma accumula dati temporanei inutilizzati; svuotarla libera spazio senza perdere dati.

💾

Spazio e cache

  • Lo spazio pieno rallenta il telefono: sistema e app hanno bisogno di spazio libero per funzionare bene.
  • La cache è dati temporanei per caricare le app più velocemente: può crescere a diversi gigabyte.
  • Android: Impostazioni → App → seleziona app → Svuota cache. iOS gestisce la cache automaticamente.
  • Prima di cancellare file: controlla cosa pesa di più in Impostazioni → Memoria → Analizza.

Trasferire a PC

Spostare file tra telefono e computer si fa via cavo USB, WiFi o cloud — ogni metodo ha velocità e casi d'uso diversi.

Metodo più veloce

WiFi: Nearby Share (Android) o AirDrop (iOS su Mac). Da Android a PC Windows: cavo USB, poi scegli "Trasferimento file". Da iPhone: Finder su Mac o cloud. Per molti file: cavo USB è il più veloce e affidabile.

🖥️

Trasferire a PC

  • USB: collega il cavo, scegli "Trasferimento file" (Android) o usa Finder/iTunes (iOS).
  • WiFi diretto: Nearby Share su Android, AirDrop su Apple, per file singoli senza cavo.
  • Cloud: carica su Drive o OneDrive dal telefono, scarica dal PC — comodo ma richiede connessione.

Condivisione file

Condividere tramite link mantiene il controllo sul file; inviare una copia cede quel controllo.

Link vs copia

Condividere via link (Drive, iCloud) non crea duplicati e permette di revocare l'accesso. Inviare il file crea una copia che non controlli più. Per documenti di lavoro preferisci sempre il link condivisibile.

🔗

Condivisione file

  • Inviare il file via WhatsApp o email crea una copia su ogni dispositivo: nessun controllo su chi la ridistribuisce.
  • Un link condiviso è revocabile, tracciabile e non duplica il file su ogni dispositivo.
  • Per file grandi, i link cloud evitano i limiti di dimensione delle chat e delle email.

File sensibili

Documenti personali fotografati non vanno lasciati nella galleria aperta: usare cartelle protette o app con lock.

Dove tenerli

Documenti sensibili (carta d'identità, codice fiscale, referti) in cartella protetta: Samsung Secure Folder, app Note iOS con lock, o app vault cifrata. Non nella galleria principale visibile a tutte le app.

🔒

File sensibili

  • La galleria foto non è un archivio sicuro: app con accesso alle foto possono vederle tutte.
  • Foto di documenti, dati bancari e referti medici richiedono protezione aggiuntiva.
  • Una cartella bloccata o cifrata è accessibile solo con PIN separato, anche se lo schermo è sbloccato.

Nomi chiari e cestino

Nomi descrittivi con data permettono di trovare i file subito; il cestino trattiene i file eliminati 30 giorni prima della cancellazione definitiva.

🗂️

Nomi chiari e cestino

  • Schema utile: AAAA-MM-GG-descrizione.ext → 2026-06-30-contratto-affitto-v2.pdf. La data in testa permette ordinamento automatico.
  • Evita nomi come "documento.pdf", "foto.jpg", "file1" — impossibili da ritrovare dopo settimane.
  • Il cestino trattiene i file eliminati fino a 30 giorni: utile per errori, non per cancellazioni definitive.
  • Cancellare dalla galleria non cancella dal cloud (Google Foto, iCloud): controlla anche lì.
